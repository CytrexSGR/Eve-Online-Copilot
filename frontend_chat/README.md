# EVE Co-Pilot Chat Frontend

Modern chat interface for EVE Co-Pilot AI with WebSocket support and audio capabilities.

## Quick Start

```bash
npm install
cp .env.example .env
npm run dev
```

Visit: http://localhost:5173

## Features

- Real-time chat with WebSocket
- Voice input (Speech-to-Text)
- Markdown message formatting
- Tool call visualization
- EVE Online dark theme
- Context management (character/region)

## Requirements

- Node.js 18+
- AI Copilot Server running on port 8001
- EVE Co-Pilot API on port 8000
